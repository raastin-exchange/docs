# لغو دسته‌ای سفارش‌ها

<div style={{textAlign: 'left'}}>

    POST /api/v1/market/orders/bulk-cancel

</div>

:::info توضیح

این API برای لغو چندین سفارش به صورت همزمان استفاده می‌شود. می‌توانید تا 100 سفارش را در یک درخواست لغو کنید.

:::

## پارامترهای Request

| نام پارامتر          | نوع   | اجباری      | توضیح                                              | مثال                          |
|----------------------|-------|-------------|----------------------------------------------------|-------------------------------|
| id_list              | Array | شرطی | لیست شناسه سفارش‌ها (یکی از id_list یا client_order_id_list الزامی است) | ["123", "456", "789"]         |
| client_order_id_list | Array | شرطی | لیست شناسه‌های سفارش توسط کاربر                   | ["order-1", "order-2"]        |

## درخواست

**لغو با استفاده از id_list:**
```json
{
    "id_list": ["1234567", "1234568", "1234569"]
}
```

**لغو با استفاده از client_order_id_list:**
```json
{
    "client_order_id_list": ["my-order-1", "my-order-2", "my-order-3"]
}
```

## پاسخ

```json
{
    "status": "done"
}
```

:::tip نکات مهم

- حداکثر 100 سفارش در یک درخواست
- فقط سفارش‌های با وضعیت NEW یا PENDING قابل لغو هستند
- اگر یک سفارش از لیست یافت نشود، بقیه همچنان لغو می‌شوند

:::
