# لیست نمادهای بازار

<div dir="ltr" style={{textAlign: 'left'}}>

    GET /api/v1/market/symbols

</div>

:::info توضیح

این API برای دریافت لیست تمام نمادهای قابل معامله در راستین استفاده می‌شود. این endpoint اطلاعات کاملی درباره هر جفت ارز شامل کارمزدها، دقت قیمت و مقدار، و محدودیت‌های معامله ارائه می‌دهد.

این API عمومی است و نیازی به احراز هویت ندارد.

:::

## پارامترهای Query (اختیاری)

می‌توانید نمادها را با استفاده از پارامترهای زیر فیلتر کنید:

| نام پارامتر      | نوع     | اجباری | توضیح                               | مثال   |
|------------------|---------|--------|-------------------------------------|--------|
| asset            | String  | False  | فیلتر بر اساس ارز                   | BTC    |
| base_asset       | String  | False  | فیلتر بر اساس ارز پایه               | IRT    |
| enable           | Boolean | False  | فقط نمادهای فعال                    | true   |
| strategy_enable  | Boolean | False  | نمادهای فعال برای استراتژی          | true   |
| margin_enable    | Boolean | False  | نمادهای فعال برای مارجین             | true   |
| include_hidden   | String  | False  | نمایش نمادهای مخفی                  | 1      |
| stats            | String  | False  | نمایش آمار بازار                    | 1      |

:::tip نکات مهم

### فیلدهای مهم:

- **tick_size:** تعداد ارقام اعشار برای قیمت
  - `tick_size=0` یعنی قیمت باید عدد صحیح باشد (مثلاً 3500000000 ریال)
  - `tick_size=2` یعنی قیمت می‌تواند 2 رقم اعشار داشته باشد (مثلاً 45.25 تتر)

- **step_size:** تعداد ارقام اعشار برای مقدار
  - `step_size=8` یعنی مقدار می‌تواند تا 8 رقم اعشار داشته باشد (مثلاً 0.00000001 بیت‌کوین)
  - `step_size=5` یعنی مقدار می‌تواند تا 5 رقم اعشار داشته باشد (مثلاً 0.12345 اتریوم)

- **taker_fee / maker_fee:**
  - Maker: کسی که سفارش limit می‌گذارد و به orderbook اضافه می‌شود
  - Taker: کسی که سفارش market می‌گذارد و از orderbook برمی‌دارد
  - مقدار به صورت اعشار است (0.002 = 0.2٪)

### ترتیب نمادها:
نمادها براساس اولویت و محبوبیت مرتب می‌شوند:
1. نمادهای Trending (asset.trend)
2. اولویت نمایش (asset.order)
3. نمادهای پایه Trending (base_asset.trend)
4. اولویت نمایش پایه (base_asset.order)

### is_bookmarked:
این فیلد فقط در صورت احراز هویت کاربر مقدار دارد. برای کاربران مهمان همیشه `false` است.

:::

:::caution نکته

- بازارهایی با `enable=false` غیرفعال هستند و نمی‌توان روی آن‌ها معامله کرد
- بازارهایی با `order_disable=true` موقتاً بسته شده‌اند (معمولاً برای تعمیرات)
- بازارهایی با `margin_order_disable=true` فقط معاملات spot امکان‌پذیر است
- با استفاده از `include_hidden=1` می‌توانید نمادهای غیرفعال را هم ببینید

:::
