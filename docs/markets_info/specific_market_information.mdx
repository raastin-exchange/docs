#   اطلاعات یک بازار خاص


<div dir="ltr" style={{textAlign: 'left'}}>

    GET /api/v1/market/symbols/&#123;symbol&#125;

</div>

:::info توضیح

این API برای دریافت اطلاعات یک بازار خاص به همراه آمار 24 ساعت گذشته استفاده می‌شود.

**پارامتر مسیر:**
- `symbol`: نماد بازار (مثال: BTCIRT، ETHUSDT، USDTIRT)

این API عمومی است و نیازی به احراز هویت ندارد.

:::

:::tip نکات مهم

### آمار 24 ساعته:

تمام فیلدهای آماری (`price`، `change`، `high`، `low`، `volume`، `base_volume`) مربوط به 24 ساعت گذشته هستند.

### تفسیر تغییرات:

- **change > 0**: قیمت افزایش یافته (صعودی)
- **change < 0**: قیمت کاهش یافته (نزولی)
- **change_percent**: درصد تغییرات نسبت به 24 ساعت قبل

**مثال محاسبه:**
```
قیمت فعلی: 3,500,000,000 ریال
تغییرات: +50,000,000 ریال
قیمت 24 ساعت قبل: 3,450,000,000 ریال
درصد تغییرات: (50,000,000 / 3,450,000,000) × 100 = 1.45%
```

### حجم معاملات:

- **volume**: حجم کل معاملات برحسب ارز معامله شده (asset)
  - برای BTCIRT: برحسب BTC
  - مثال: "15.5" یعنی 15.5 بیت‌کوین معامله شده

- **base_volume**: حجم کل معاملات برحسب ارز پایه (base_asset)
  - برای BTCIRT: برحسب ریال
  - مثال: "54250000000" یعنی 54.25 میلیارد ریال معامله شده

### مقایسه با endpoint کلی:

- **این endpoint:** `/api/v1/market/symbols/{symbol}` - فقط یک بازار خاص
- **Endpoint کلی:** `/api/v1/market/symbols?stats=1` - لیست همه بازارها با آمار

برای نمایش لیست بازارها با آمار، از endpoint کلی استفاده کنید.

:::

:::caution نکته

اگر بازاری معامله فعالی در 24 ساعت گذشته نداشته باشد، برخی از فیلدها (مانند `change`, `volume`) ممکن است `"0"` یا `null` باشند.

:::
