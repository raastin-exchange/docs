#   احراز هویت و توکن

API ها در راستین به دو دسته تقسیم می‌شوند. دسته نخست API های عمومی است که نیازی به احراز هویت و ارسال توکن نیست. برای دسترسی به API های خصوصی لازم است توکن به عنوان HTTP Header درخواست به صورت زیر ارسال شود.

<div dir='ltr'>

```Authorization: Token YOUR_TOKEN```

</div>

---

:::caution محدودیت فراخوانی

- **محدودیت لحظه‌ای**: 60 درخواست در دقیقه
- **محدودیت روزانه**: 86,400 درخواست در روز (برای API Token)

برای اطلاعات بیشتر، صفحه [محدودیت‌های API](/docs/rate_limits) را مطالعه کنید.

:::

:::info توجه
جهت احراز هویت از این HTTP Header استفاده کنید و به خاطر داشته باشید که **YOUR_TOKEN** را با توکن خود جایگذاری کنید.

:::

:::tip نکته

برای دریافت توکن می‌توانید با مراجعه به پنل کاربری خود در راستین از بخش حساب کاربری وارد صفحه <a href="https://raastin.com/account/api-management">مدیریت API</a> شده و توکن خود را دریافت نمایید.

:::
