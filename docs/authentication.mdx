# احراز هویت

APIها در <ExchangeName /> به دو دسته تقسیم می‌شوند. دسته نخست، **APIهای عمومی** هستند که برای استفاده از آن‌ها نیازی به احراز هویت یا ارسال توکن وجود ندارد. برای دسترسی به **APIهای خصوصی**، لازم است توکن احراز هویت به‌عنوان HTTP Header در درخواست‌ها، مطابق ساختار زیر، ارسال شود.

<div dir='ltr'>

```Authorization: Token YOUR_TOKEN```

</div>

---


:::info توجه
جهت احراز هویت از این HTTP Header استفاده کنید و به خاطر داشته باشید که **YOUR_TOKEN** را با توکن خود جایگذاری کنید.

:::

:::tip نکته

برای دریافت توکن می‌توانید با مراجعه به پنل کاربری خود در <ExchangeName /> از بخش حساب کاربری وارد صفحه مدیریت API شده و توکن خود را دریافت نمایید.

:::
