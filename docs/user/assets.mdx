# دارایی‌ها و ارزها

## لیست دارایی‌ها

<div style={{textAlign: 'left'}}>

    GET /api/v1/assets

</div>

:::info توضیح

این API برای دریافت لیست تمام ارزهای دیجیتال و فیات قابل استفاده در راستین استفاده می‌شود. این endpoint اطلاعات کاملی درباره هر ارز شامل موجودی، قیمت، امکان واریز و برداشت ارائه می‌دهد.

:::

## جزئیات یک دارایی

<div style={{textAlign: 'left'}}>

    GET /api/v1/assets/&#123;symbol&#125;

</div>

:::info توضیح

این API برای دریافت اطلاعات جزئی یک ارز خاص استفاده می‌شود.

**پارامتر مسیر:**
- `symbol`: نماد ارز (مثال: BTC، ETH، USDT)

:::

:::tip نکات مهم

### تفاوت با `/api/v1/wallets`:

**Assets API (`/api/v1/assets`):**
- لیست همه ارزهای موجود در پلتفرم
- حتی ارزهایی که کاربر موجودی ندارد نمایش داده می‌شود
- شامل اطلاعات عمومی ارزها (trending، market availability)
- برای نمایش لیست کامل ارزها مناسب است

**Wallets API (`/api/v1/wallets`):**
- فقط کیف پول‌های کاربر
- معمولاً فقط ارزهایی که کاربر موجودی دارد
- تمرکز بر اطلاعات مالی (balance، locked)
- برای مدیریت دارایی‌های کاربر مناسب است

### فیلدهای مهم:

- **precision / step_size:** تعداد ارقام اعشار مجاز
  - استفاده در محاسبات و نمایش
  - مثلاً برای BTC: `0.12345678` (8 رقم اعشار)

- **trend:** ارزهای پرطرفدار که در صفحه اصلی برجسته نمایش داده می‌شوند

- **market_irt_enable:** آیا بازار ریالی (مثلاً BTCIRT) برای این ارز وجود دارد

- **trade_enable:** آیا این ارز قابل معامله است
  - false برای ارزهای در حال توسعه یا غیرفعال

- **pin_to_top:** برای کاربران احراز هویت شده، ارزهایی که در بالای لیست شخصی نمایش داده می‌شوند

:::

:::caution نکته

این API برای دریافت لیست ارزهاست. برای دریافت **قیمت لحظه‌ای** ارزها از API بازارها (`/api/v1/market/symbols?stats=1`) استفاده کنید.

:::
