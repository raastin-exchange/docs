# دریافت آدرس واریز


<div dir="ltr" style={{textAlign: 'left'}}>

    GET /api/v1/deposit/address/

</div>

برای دریافت آدرس واریز ارز دیجیتال از این API استفاده کنید. با مشخص کردن مقادیر `coin` و `network` در query params، در صورت وجود کیف پول آدرس آن برگردانده می‌شود و در صورت عدم وجود، کیف پول ساخته شده و آدرس آن برگردانده می‌شود. برای استفاده از این API لازم است توکن احراز هویت در header ارسال شود.

:::info نکته
کاربر باید حداقل سطح 2 احراز هویت داشته باشد.
:::

## پارامترهای Query

| نام پارامتر | نوع    | اجباری | توضیح                           | نمونه   |
|-------------|--------|--------|--------------------------------|---------|
| coin        | String | بله    | نماد ارز دیجیتال مورد نظر       | "USDT"  |
| network     | String | بله    | نماد شبکه انتقال               | "TRC20" |

## ساختار پاسخ

| نام پارامتر | نوع    | توضیحات                                        | نمونه                                        |
|-------------|--------|------------------------------------------------|----------------------------------------------|
| address     | String | آدرس واریز                                     | "TYDzsYUEpvnYmQk4zGP9sWWcTEd2MiAtW6"        |
| memo        | String | ممو یا تگ (برای شبکه‌هایی که نیاز دارند)       | "123456789"                                  |

:::tip نکات مهم

### شبکه‌های مختلف:

برای هر ارز، باید شبکه مناسب را انتخاب کنید:

- **USDT**: شبکه‌های TRC20، ERC20، BEP20
- **BTC**: شبکه Bitcoin
- **ETH**: شبکه Ethereum

### ممو (Memo/Tag):

برخی شبکه‌ها مانند XRP، XLM و TON نیاز به ممو یا تگ دارند. در این موارد حتماً هنگام واریز، ممو را وارد کنید وگرنه دارایی از دست می‌رود.

:::

:::caution هشدار

- همیشه قبل از واریز، شبکه را با دقت بررسی کنید
- واریز به آدرس اشتباه یا شبکه نامناسب غیرقابل بازگشت است
- برای هر ارز و شبکه، یک آدرس منحصربه‌فرد صادر می‌شود

:::
