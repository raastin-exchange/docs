# دریافت مقادیر آزاد هر کیف پول

<div dir="ltr" style={{textAlign: 'left'}}>

    GET /api/v1/wallet/balance

</div>

:::info توجه

برای دریافت لیست کیف پول ها و مقادیر آزاد هر کدام میتوانید از این api استفاده کنید.

:::

## ساختار پاسخ

پاسخ به صورت آرایه‌ای از کیف پول‌ها برگردانده می‌شود:

| نام پارامتر | نوع    | توضیحات                  | نمونه       |
|-------------|--------|--------------------------|-------------|
| asset       | String | نماد ارز دیجیتال         | "USDT"      |
| free        | String | موجودی آزاد (قابل برداشت)| "100.5"     |
| balance     | String | موجودی کل                | "150.0"     |
| locked      | String | موجودی قفل شده در سفارش  | "49.5"      |
