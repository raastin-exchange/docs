# دریافت مقادیر آزاد هر کیف پول

<div dir="ltr" style={{textAlign: 'left'}}>

    GET /api/v1/wallet/balance/

</div>

برای دریافت فهرست کیف‌پول‌ها و میزان موجودی آزاد هر یک، می‌توانید از این API استفاده کنید.

برای استفاده از این API، لازم است توکن احراز هویت در بخش **Header** درخواست ارسال شود.

## ساختار پاسخ

پاسخ به صورت آرایه‌ای از کیف پول‌ها برگردانده می‌شود:

| نام پارامتر | نوع    | توضیحات                  | نمونه            |
|-------------|--------|--------------------------|------------------|
| asset       | String | نماد ارز دیجیتال         | "USDT"           |
| free        | Number | موجودی آزاد (قابل برداشت)| 100.5            |
| balance     | String | موجودی کل                | "150.00000000"   |
| locked      | String | موجودی قفل شده در سفارش  | "49.50000000"    |

:::tip نکات مهم

### محاسبه موجودی:
```
موجودی کل (balance) = موجودی آزاد (free) + موجودی قفل شده (locked)
```

### تفاوت با /api/v1/wallets/:
- این endpoint فقط اطلاعات موجودی را برمی‌گرداند
- سریع‌تر است و اطلاعات کمتری شامل می‌شود
- `free` به صورت عدد (Number) برگردانده می‌شود در حالی که `balance` و `locked` رشته هستند

:::
