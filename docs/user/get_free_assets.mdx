# دریافت مقادیر آزاد هر کیف پول

<div style={{textAlign: 'left'}}>

    GET /api/v1/wallet/balance

</div>

:::info توجه

برای دریافت لیست کیف پول ها و مقادیر آزاد هر کدام میتوانید از این api استفاده کنید.

:::

## ساختار Response

| نام پارامتر | نوع   | توضیحات     |
|-------------|-------|-------------|
| wallet      | Array | کیف‌های پول |

**ساختار wallet:**

| نام پارامتر | نوع    | توضیحات    | نمونه    |
|-------------|--------|------------|----------|
| asset       | String | ارزدیجیتال | "USDT"   |
| free        | String | مقدار آزاد | "100000" |
